  • To delete all nodes and relationships:

    MATCH (n)
    OPTIONAL MATCH (n)-[r]-()
    DELETE n,r;

Part two

  • Cypher refcard: http://neo4j.com/docs/stable/cypher-refcard/

    • Creating databases

    Import from a CSV: LOAD CSV WITH HEADERS FROM "xxxx.csv" AS line

    To create a node: CREATE (x:Label {property:Property})

    To create a relationship: CREATE (x)-[r:CONNECTED_TO]->(y)
